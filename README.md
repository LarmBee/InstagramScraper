"# InstagramScraper" 
The scraper is written in Python and Selenium
Currently all the scraper gets you is :
1:The Name
2:Number of posts 
3.Number of Followers 
4.Number Following 
5.Post links
6.Are the posts videos or images
7.The number of likes on each post
